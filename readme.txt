An essential integration addon for Fluent Forms.

== Description ==

Logic Essentials Fluent Forms Addon is a powerful integration that extends Fluent Forms functionality.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/logic-essentials-fluent-forms-addon` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Logic Essentials screen to configure the plugin.

== Frequently Asked Questions ==

= Does this require Fluent Forms? =
Yes, you must have Fluent Forms installed and activated to use this addon.
